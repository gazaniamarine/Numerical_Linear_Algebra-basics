# 05 — Eigenvalues

Iterative algorithms for finding eigenvalues and eigenvectors.

## 📋 Contents

- **Power Iteration**: Finding the dominant (largest magnitude) eigenvalue.
- **Inverse Iteration**: Finding the eigenvalue closest to a specified shift $\mu$.
- **QR Algorithm**: Iterative method for finding all eigenvalues of a matrix.

## 🚀 Examples

- `eigenvalue_example.py`: Comparing the convergence of Power and Inverse iteration.
