# 10 — Applications

Real-world applications of numerical linear algebra algorithms.

## 📋 Contents

- **Image Compression**: Using SVD to reduce image storage while preserving visual quality.
- **PageRank**: Implementing the Google Search ranking algorithm via power iteration on transition matrices.
- **Principal Component Analysis (PCA)**: Dimensionality reduction and data projection using singular vectors.

## 🚀 Examples

The scripts in this directory are the applications themselves:
- `image_compression_svd.py`
- `pagerank.py`
- `pca_example.py`
